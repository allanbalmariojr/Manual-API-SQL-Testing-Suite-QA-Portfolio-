# 🧪 Manual, API & SQL Testing Suite (QA Portfolio)

This repository contains my hands-on QA testing work covering **manual testing, API testing, SQL validation, mobile testing, and end-to-end system testing**.

It demonstrates structured QA practices across UI, backend, and database layers using real-world testing techniques.

## 🎯 Project Purpose

This project showcases my ability to:

- Perform end-to-end testing across UI, API, and database layers
- Document defects using structured QA reporting (Jira-style)
